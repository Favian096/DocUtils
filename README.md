# 基于Apache POI的简单封装

## 实现对Excel文档简单读写

- 对Excel指定表格输入行列号读取数据(.xls|.xlsx)
-
- 对Excel指定表格位置写入数据(.xlsx)
    - 设置单元格表框
    - 设置单元格对齐方式
    - 合并单元格
    - Excel表格转PDF

## 实现对Word文档的内容简单读取和数据简单写入(.docx)

- 对Word文档读取内容输出为List<T>
    - 段落, 表格, 图片, 页眉页脚

- 对Word文档写入数据
    - 写入段落, 生成表格
    - 自定义对齐方式
    - Word文档转PDF
